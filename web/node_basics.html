How does require() work ?

require() 源码解读 by 阮一锋
nodejs official reference about module

  1. Require statements are executed sychronously. That is, all the statements in the required module are executed before continuing execute statements after the require statement.
  2. Required modules are cached. They can be viewed as single instance.

Install

npm install installs all dependent packages (and those dependent on these packages, and…) specified in package.json.
npm install <package> installs ‘package’.

options:

  • -g : install globally (default locally in ./node_modules/)
  • –save: save to package.json as dependency
  • –save-dev: save to package.json as dependencyDev
