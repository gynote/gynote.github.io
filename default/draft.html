Unicode

尽可能选择utf-8, 当作单字节处理, 不要转成u32, u16. regex_replace 一个utf-8字符可以通过regex_replace 多个\x字符来完成.
由utf-8的编码规则可知, 任一ASCII中的字符的编码都不可能是其它任意字符编码的一部分. 可以说, utf-8编码是prefix, infix, postfix 编码. 因此, 处理utf-8字符时完全可以当作多个单字节来处理.
c++中\u好像只能用在u32string, u16string中, 不能用于utf-8. utf-8可以用\x. \x 可以出现在任意字符串中, 但 \U和\u只能出现在 u16“”, u32“” 中. utf-8, utf-16, utf-32 对应的c++ type 分别是unsigned char, unsigned short, unsigned int (int_32)
